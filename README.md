# AiConverter
usage: AiConverter {zones|targets(Incomplete)} csvfile.csv [output]
Example CSV and template can be found with the executable
Make sure your file is a .csv file and not another spreadsheet format like .xlsx
Zones values should be even or they will be incorrect in game.");
Zones Shapes:
    0, r, rect, rectangle                         = rectangle
    1, tul, triupleft, triangle up left           = triangle, corner in upper left
    2, tur, triupright, triangle up right         = triangle, corner in upper right
    3, tbl, tribotleft, triangle bottom left      = triangle, corner in bottom left
    4, tbr, tribotright, rectriangle bottom right = triangle, corner in bottom right
